Chicken and Corn Game API

In this game you need to supply or modify the JavaScript implementation of the algorithm for each player piece to behave by. To help you do this there are a number of APIs that you can call in your script.

Each implementation of an algorithm is provided an object called Me that is of type Piece that presents the player piece that is being asked for a move. With this object you determine your position on the game board by getting its position object. You can use the other object to locate and compute distances to your target. Eventually if you are coding a human algorithm you want to get your piece next to a chicken so you can then call the capture() method to capture the chicken. If you are coding the algorithms for the chickens, you'll want to get as far away from the humans as you can.

A position object holds the row and col values of the board location. The game board coordinates (row and col) have values 0 to 7.

Globals

API Summary

_Game

memberdescription
name()Returns a simple string with the name of the game.
availableMoves(Piece)Returns an array of move strings ("U", "D", "L", "R") indicating the allowable moves for the piece on the gameboard at the moment.
availableMoves(row,col)Returns an array of move strings ("U", "D", "L", "R") indicating the allowable moves for any piece at position (row,col).
isOccupied(Position)Returns true if the position passed in is currently occupied.
isOccupied(row,col)Returns true if the position identified by the row and column is currently occupied.
log(string)Writes a string to the console visible under the gameboard on the UI.
moves()Returns the current number of moves made so far.
randomInt(min, max)Returns a uniformly distributed random integer (using the JavaScript Math.random() function) between the min and max (inclusive).

Piece

memberdescription
name The name property of the piece, of type String.
position The property with the current position of the piece, of type Position.
capture(Piece) The human piece move on top of a chicken (capturing it). This can only happen if the human piece (Farmer or Wife) is already next to a chicken.
distanceTo(Position)Returns the distance of this piece to the position position object passed in as the argument.
distanceTo(row,col)Returns the distance of this piece to the position indicated by the row and col.
isActive()Returns true if this piece is still chasing a chicken. Once a piece has captured a chicken it is no longer active and remains in the last position (with the chicken).
isNextTo(Piece)Returns true if this piece is next to the piece in the argument. There are four globally defined pieces; Rooster, Farmer, Wife and Hen.
moveRandom()Returns a valid move for this piece that is randomly selected from all the available moves for this piece.

Position

memberdescription
row The current row that this piece is in, of type number (integer).
col The current column that this piece is in, of type number (integer).
isNextTo(Position) Returns true if the current position is next to the position object passed in as the argument.
